## Run Locally

Clone the project

```bash
  git clone https://github.com/Emin14/jadoo-react.git
```

Go to the project directory

```bash
  cd jadoo-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Features

- Switch language
- Playing a YouTube video on a page
- Slider on page
